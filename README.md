# Counter Toolkit

Counter Toolkit is a Flutter app scaffold for front-counter postal work.

The first build is focused on a practical toolkit for:

- parcel tracking and trace lookup
- size and weight guidance
- service and format checks
- quick counter-side reference tools

## Current State

- multi-platform Flutter project scaffolded
- starter dashboard in place instead of the default Flutter counter demo
- user-facing labels updated for Counter Toolkit branding

## Development

```bash
flutter pub get
flutter analyze
flutter test
flutter run -d macos
```
